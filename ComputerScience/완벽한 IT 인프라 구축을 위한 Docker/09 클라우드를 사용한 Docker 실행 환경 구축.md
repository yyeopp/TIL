# Chapter 09: 클라우드를 사용한 Docker 실행 환경 구축

---

Docker는 이미지만 있으면 동일한 인프라 환경에서 어플리케이션을 가동시킬 수 있다.

이를 통한 어플리케이션의 **높은 이식성**이 최대의 강점이다.

- GCP의 매니지드 서비스인 Google Kubernetes Engine (GKE)를 사용해 Docker 실행 환경을 구축하는 것이 하나의 예시가 될 것.
